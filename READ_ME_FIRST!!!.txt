Note scripts 1 and 2 should work on all moto devices and 3 on most droid devices

1-rsd-flash-servicefile-windows.bat
run this script to convert the servicefile.xml in you motorola firmware to a batch script that will not wipe your device data

2-rsd-flash-flashfile-windows.bat
run this script to convert the flashfile.xml in you motorola firmware to a batch script that WILL wipe your device data

3-rsd-flash-Droid-windows.bat
run this script to convert xml files from older device that do not have a servicefile or flashfile but have a xml file that is he model of the device. if the script that is created doesn't work then use 2-rsd-flash-flashfile-windows.bat and see if it works.
